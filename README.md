A fun, interactive web app built to help users discover their purr-fect cat preferences. Swipe right on cats you like and left on those you don't! This single-page application features a smooth, mobile-first interface and pulls adorable cat pictures directly from the Cataas.

Features:

Tinder-style Swiping: Intuitive swipe gestures for liking and disliking cat images.

Mobile-First Design: Optimized for a seamless experience on mobile devices.

Results Summary: View a final summary of all your liked cats.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
